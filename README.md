**Важно:** </br>

## Съдържание

- **Тема 1-2:** Увод. Примитивни типове данни. Оператори.
- **Тема 3:** Условни конструкции - if, switch и тернарен оператор.
- **Тема 4:** Цикли – while, do-while и for. Оператори break и continue.
- **Тема 5:** Функции.
- **Тема 6:** Масиви.
- **Тема 7:** Подготовка за контролно.
