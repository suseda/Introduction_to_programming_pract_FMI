## Примери - теория
**1.** Каква е разликата в изхода от двете парчета код?
```c++
#include <iostream>

int main()
{
	int x = 0;
	while (x > 0)
	{
		std::cout << "x";
		x--;
	}
}
```
```c++
#include <iostream>

int main()
{
	int x = 0;
	do
	{
		std::cout << "x";
		x--;
	} while (x > 0);
}
```
**2.** Какво ще отпечата следния код?
```c++
#include <iostream>

int main()
{
        int q = 5;
  	while (q = 6)
  	{
  		if (q == 5)
  		{
  			break;
  		}
  		std::cout << q;
  		q--;
  	}
}
```
